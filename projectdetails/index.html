The Crossroads of Film

Project Description

For our project, we are building a movie recommendation/search service. It is a movie organizer that tracks movie data including actors, ratings, awards, etc. that will give recommendations based on different measures of similarity from multiple sources and provide users with the ability to access and filter this movie information. Users will have the ability create an account and like different movies. Users will also be able to follow other users and get recommendations for movies to watch based on the likes of users they follow.

Usefulness

Often, when thinking about or discussing movies, you'd like to be able to search for movies by various fields (actors, time-frame, directors, etc). Additionally, you'd like to get recommendations for movies based on movies that your friends enjoy. We'd like to build a service that fufills all our movie search/recommendation needs. Rather than having to use IMBD to get movie data then a different form of social media to get recommendations, we want to bundle it all into one helpful web application. Ideally, we want our service to allow users to search for movies and actors in creative ways. For example, if your movie night is centered around watching an adventure movie that has won best picture, then you will be able to pull up a list of movies that fit those specifications.

Realness

The majority of our movie data will come from the IMDb dataset (http://www.imdb.com/interfaces). We will use the movie data, actor/actresses data, genre data, ratings data, and "appears in" data from IMBD. We will then use user data from external users including movie "likes" and follower data.

Functionalities

Basic Functions

Inserting: Our web application will consist of a login where a user can create an account which inserts credentials into the Users database. The user will also be able to insert a new movie if it is not already included in the database by using the "Insert" button on the lower right hand side of the homepage.
Updating: Our web application will allow users to rate movies. This will update the rating to account for the additional rating.
Searching: The user will be able search for a movie which will return a list of all the movies that contain the search criteria in the name. The user can also search on actor to return all movies with that actor. This query consists of a join on the Movies and Actors tables.

Advanced Functions

Advanced Search: This will provide the user the ability to craft their own creative searches. We provide some examples below to illustrate use cases.

  1. Search for all movies with Denzel Washington and Clive Owen between 2000 and 2010.
  2. Search for the highest rated romantic comedies within the past 10 years.
  3. What are the lowest rated movies in the past 10 years with Anne Hathaway.
  4. Search for high rated war movies with specific actors.

Collaborative Filtering: This will allow users to log in to the site and 'like' different movies. The user will also be able to follow different users and get recommendations of movies to watched based on the 'likes' of the users they are following.
